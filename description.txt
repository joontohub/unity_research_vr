작은 유니티짱에 애니메이션을 적용해 보았습니다. 

애니메이션은 Drink, Eat, Blink 가 있습니다. 

A 키를 누르면, 책상 앞에 있는 LIGHT 를 껐다 켰다 할 수 있습니다. 

B 키를 누르면, 눈을 깜빡이고 주변을 둘러보는 애니메이션이 재생됩니다. 

E 키를 누르면, 먹는 애니메이션이 시작됩니다. ( 2초뒤에 애니메이션이 자동 종료됩니다.)

R 키를 누르면, 수동으로 먹는 애니메이션을 끌 수 있습니다. 

D 키를 누르면, 마시는 애니메이션이 재생됩니다. (2초 뒤에 애니메이션이 자동 종료됩니다. )

F 키를 누르면, 수동으로 마시는 애니메이션을 종료할 수 있습니다.  

L 키를 누르면, 기본 상태 -> 마시기 -> 둘러보기 -> 먹기 -> 둘러보기 ( 유지 ) 가 됩니다. 

M 키를 누르면, 음악이 처음부터 재생됩니다. 

N 키를 누르면, 음악을 수동으로 끌 수 있습니다.
